# how to build an env with virtualenv in python
 ## install virtualenv
- three method
  
  `$ sudo easy_install virtualenv` 
  
  `$ sudo pip install virtualenv` 
  
  `$ sudo apt-get install python-virtualenv`
  
- build an env 
  
    `$ virtualenv venv(your env name)` 
  
- activate your env
  
    `source venv/bin/activate`

- deactivate your env 

    `deactivate`
